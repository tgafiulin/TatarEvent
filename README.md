# TatarEvent

## Privacy Policy

This Chrome extension, “TatarEvent 2.0”, does not collect, store, or transmit any personal data about you.
The extension only modifies the appearance of pages on the jevent.jugru.org website in your browser in order to hide other reviewers’ scores and comments. All processing happens locally in your browser and no information is sent to any external servers.

If you have any questions, contact: t.gafiulin@yandex.ru.
